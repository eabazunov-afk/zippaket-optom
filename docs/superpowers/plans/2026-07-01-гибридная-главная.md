# Гибридная главная (витрина-поиск на базе тёмного редизайна) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Превратить имиджевый лендинг главной (`master`, тёмная тема) в продающую витрину-поиск: hero с подбором по размеру/толщине/типу замка, секции «Хиты продаж» и «Новинки» с ценами и остатками, открытые опт-условия и единые контакты — сохранив крепкую базу тёмной версии.

**Architecture:** Данные берутся из БД через существующий класс `Catalog` (`includes/catalog_functions.php`). Вся чистая логика отбора и форматирования выносится в новый `includes/home_view.php` и покрывается PHPUnit-тестами (без БД, как принято в проекте). Разметка секций собирается в `index.php` из этих хелперов. Поиск с главной ведёт GET-запросом в существующий каталог `/katalog_zip_paketov/`, который уже умеет фильтровать.

**Tech Stack:** PHP 8.3, MySQL 8.x (PDO), PHPUnit 9.6, ванильный JS. Тёмная тема — `css/site-premium.css` + `css/home.css`. Без новых зависимостей.

## Global Constraints

- PHP 8.3, запуск тестов: `php vendor/phpunit/phpunit/phpunit --bootstrap tests/bootstrap.php tests` из `www/` (ожидается `OK`).
- Тесты — на чистых функциях, БД в тестах НЕ трогаем (паттерн проекта: тестируем форматтеры/логику отбора на массивах в памяти).
- Тексты — на русском, тон делового опта (без «воды»).
- Данные товаров — только из БД (`products`, `is_active=1`, `price_rub>0`). Никаких захардкоженных массивов товаров.
- Контакты, телефон, email — только из констант `config.php` (`SUPPORT_PHONE`, `ADMIN_EMAIL`, `SITE_NAME`). Плейсхолдеров в разметке быть не должно.
- Опт-условия карточки: цена + оптовая градация (розница / опт −8% / опт −18%, как в текущем `z_card`), `min_order_qty`, `qty_step`, остаток.
- Тёмная тема — базовая. Светлый коммерческий вариант для A/B — ОТДЕЛЬНЫЙ будущий план, здесь не делаем.
- Поля товара, доступные из БД: `id, category, full_name, short_name, width, height, thickness, color, features, unit, min_order_qty, qty_step, quantity_sold, stock_quantity, price_rub, image_url, created_at`.

```php
<?php
/**
 * Чистые view-хелперы главной. Без БД и без вывода — только форматирование
 * и отбор, чтобы можно было покрыть тестами (см. tests/HomeViewTest.php).
 */

/** Цена уровня из базовой цены и множителя: "3 849,40". */
function home_price(float $base, float $mult): string {
    return number_format($base * $mult, 2, ',', ' ');
}

/** Размер из мм в "25 × 30 см". Пустая строка, если размеров нет. */
function home_size(?int $w, ?int $h): string {
    $fmt = function ($mm) {
        $cm = $mm / 10;
        return rtrim(rtrim(number_format($cm, 1, '.', ''), '0'), '.');
    };
    if (!$w || !$h) return '';
    return $fmt($w) . ' × ' . $fmt($h) . ' см';
}

/** Новинки: по created_at убыванию, первые $limit. Не мутирует вход. */
function home_pick_new(array $products, int $limit = 4): array {
    usort($products, fn($a, $b) => strcmp((string)($b['created_at'] ?? ''), (string)($a['created_at'] ?? '')));
    return array_slice($products, 0, $limit);
}

/** Хиты: по quantity_sold убыв., при равенстве — по stock_quantity убыв. */
function home_pick_hits(array $products, int $limit = 8): array {
    usort($products, function ($a, $b) {
        $s = (int)($b['quantity_sold'] ?? 0) <=> (int)($a['quantity_sold'] ?? 0);
        if ($s !== 0) return $s;
        return (int)($b['stock_quantity'] ?? 0) <=> (int)($a['stock_quantity'] ?? 0);
    });
    return array_slice($products, 0, $limit);
}
```

```php
    /**
     * Получить новинки — самые свежие активные товары с ценой.
     */
    public function getNewProducts($limit = 4) {
        try {
            $sql = "SELECT * FROM products
                    WHERE is_active = 1 AND price_rub > 0
                    ORDER BY created_at DESC, id DESC
                    LIMIT " . (int)$limit;
            $stmt = $this->db->query($sql);
            return $stmt->fetchAll();
        } catch (PDOException $e) {
            error_log("Ошибка получения новинок: " . $e->getMessage());
            return [];
        }
    }
```
