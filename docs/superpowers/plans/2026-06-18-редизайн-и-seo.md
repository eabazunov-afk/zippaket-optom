# Premium-редизайн пути покупки + SEO — Implementation Plan

**Goal:** Поднять premium-восприятие и SEO коммерческого пути (каталог → товар → корзина → checkout → успех), не трогая legacy-разметку главной. Объём согласован с заказчиком: **Коммерция + SEO** (главная — отдельной итерацией).

**Architecture:** Премиум-слой как **тема, подключаемая по флагу** (`<body class="premium">` + `css/premium.css`) — затрагивает только коммерс-страницы, не ломая главную/админку и существующие `style.css`/`catalog.css`. Дизайн-токены (палитра/радиусы/тени/кнопки) — CSS-переменные по дизайн-спеку. SEO-хелперы (`includes/seo.php`) — чистые функции (база URL, BreadcrumbList) с тестами; динамический `sitemap.php` отдаётся по чистому `/sitemap.xml` через rewrite (т.к. robots закрывает `*.php`).

**Tech Stack:** PHP 8.3.31, PDO/MySQL 8.0, PHPUnit 9.6, CSS (Inter, font-awesome).

## Global Constraints

- Премиум только на коммерс-страницах (флаг `body.premium`); главная не затрагивается.
- Не ломать существующую разметку/классы — стили накладываются поверх.
- SEO-хелперы — чистые, тестируемые; sitemap доступен по индексируемому URL.
- ЧПУ и canonical — на боевой домен (`SITE_URL`).

## File Structure

- `www/css/premium.css` — дизайн-токены (палитра `#0B1F3A`/мята `#5FE3D0`/CTA `#FFB020`),
  премиум-кнопки, карточки, стеклянная шапка `.pm-pagehead`, формы, корзина, итоги.
- `www/includes/seo.php` — `seo_base`, `seo_url`, `seo_breadcrumb_jsonld` (чистые).
- `www/sitemap.php` — динамический sitemap (главная, статика, категории, товары из БД).
- `www/.htaccess` — `RewriteRule ^sitemap\.xml$ sitemap.php`; `router.php` — то же для dev.
- Изменены: `product.php`, `katalog.php`, `cart.php`, `checkout.php`, `order_success.php`
  (подключение премиума + BreadcrumbList/canonical), `config.php` (фикс `SITE_URL`).
- `www/tests/SeoTest.php` — тесты SEO-хелперов.
